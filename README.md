# Portfolio - Lucas DJT

Bienvenue sur le dépôt du portfolio de Lucas DJT.

Ce projet est une page web dynamique réalisée avec React, permettant de présenter mn profil.

## Installation

```bash
npm i
npm start

npm run predeploy
npm run deploy
```
